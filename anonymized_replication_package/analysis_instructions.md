# Independent Replication Instructions

You are tasked with analyzing the transmission dynamics of "Pathogen X" using historical line lists to inform the public health response to a recent cruise ship outbreak. Please independently write the code and run the models to calculate the parameters below based on the provided datasets in the `data/` folder.

## Datasets Provided:
1. `data/village_a_cases.csv`: Transmission network with exact infector-infectee pairs from a historical cluster.
2. `data/village_b_cases.csv`: Line list with bounded exposure dates and symptom onset dates from a larger historical outbreak.
3. `data/cruise_cases.csv`: Current outbreak timeline to be evaluated.

## Tasks to Complete:

**1. Incubation Period Modeling (MLE)**
Using the bounded exposure dates in `village_b_cases.csv`, perform an interval-censored Maximum Likelihood Estimation (MLE) to fit a Lognormal distribution to the human-to-human incubation period. Calculate the median and Interquartile Range (IQR). 

**2. Infectiousness Profile (Generation Time Deconvolution)**
Using the exact infector-infectee pairs and symptom onset dates in `village_a_cases.csv`:
- Calculate the empirical serial intervals.
- Fit a distribution to the generation time.
- Deconvolute this against your derived incubation period to map the infectiousness profile relative to symptom onset.
- Calculate what percentage of transmission occurs in the presymptomatic/prodromal phase (before symptom onset).

**3. Transmission Heterogeneity (Negative Binomial Regression)**
Extract the offspring distribution (number of secondary cases generated by each index case) from both `village_a_cases.csv` and `village_b_cases.csv`. Fit a Negative Binomial distribution to the combined offspring data to estimate:
- The basic reproduction number ($R_0$)
- The dispersion parameter ($k$)

**4. Outbreak Extinction Probability (Branching Process Simulation)**
Using your derived $R_0$ and $k$ parameters, conduct a stochastic branching process simulation (e.g., 50,000 iterations) starting with a single introduction (like the index case on the cruise ship). 
- Calculate the probability of immediate stochastic extinction (where the outbreak generates 0 secondary cases).
- Characterize the distribution of outbreak sizes if extinction does not occur early.

Please generate your results and plot the corresponding figures, then compare them against the `replication_results.md` provided to verify accuracy.
